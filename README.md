# Sistema web para consórcio de veículos
Este projeto consiste em um website para gerenciamento das ações de um consórcio de veículos hipotético. A aplicação foi desenvolvida utilizando o framework back-end Django, e tem a funcionalidade de realizar sorteios de veículos entre os usuários cadastrados no sistema e exibir o último veículo sorteado assim como o ganhador na página principal do sistema. Trabalho feito para a disciplina de Tópicos Especiais de Sistemas de Informação II referente ao curso de Bacharelado em Sistemas de Informação pela Universidade Federal do Acre (UFAC).
